# Authentication Testing Tool

A mobile-friendly authentication testing tool for authorized security assessments.

## Download

### Android APK
- Go to Actions tab
- Download latest APK from artifacts
- Install on your Android device

### Web App
- Visit: https://[your-username].github.io/auth-testing-tool/
- Add to home screen for app-like experience

### Termux (Android)
1. Install Termux
2. Run: `pkg install python git`
3. Clone: `git clone [repo-url]`
4. Install: `pip install requests`
